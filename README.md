# 原神祈愿记录

## 功能

- [X] 将爬取结果保存到 `<uid>.json` 文件中（原神官方祈愿记录只保留六个月，获取到新的祈愿记录时，会与本地数据合并后保存）
- [X] 以 HTML 形式粗略展示祈愿记录
- [ ] 选择是否输出4星祈愿记录
- [ ] 输入 url 后自动获取祈愿记录（Android、iOS）
- [ ] 通过加载本地记录自动获取祈愿记录（PC）
